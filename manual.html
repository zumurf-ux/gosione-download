고시ONE 사용자 매뉴얼

고시ONE 앱 설치부터 라이선스 인증, 객실 관리, 월세 관리, 입퇴실 기록, 백업과 복원까지 기본 사용 방법을 안내합니다.

버전 5.0.3 기준 작성일 2026-06-15
앱 설치와 첫 실행 라이선스 인증 객실 관리 월세 관리 입퇴실 관리 백업과 복원

1. 앱 설치와 첫 실행

  1. 전달받은 APK 파일을 휴대폰에 설치합니다.
  2. 설치 후 고시ONE 앱을 실행합니다.
  3. 알림 권한 요청 창이 나오면 허용을 누릅니다.
  4. 라이선스 인증 화면이 나오면 구매한 라이선스키를 입력합니다.
  5. 인증이 완료되면 고시ONE 홈 화면으로 이동합니다.
알림을 허용하지 않으면 월세 납부 예정 알림이 오지 않을 수 있습니다. 실수로 거절했다면 휴대폰 설정에서 고시ONE 앱 알림 권한을 다시 켜주세요.

2. 라이선스 인증

라이선스키는 앱을 사용하기 위한 인증번호입니다.

  1. 앱 실행 후 라이선스 인증 화면에서 라이선스키를 입력합니다.
  2. 인증하기 버튼을 누릅니다.
  3. 인증이 완료되면 앱을 사용할 수 있습니다.

라이선스키는 재설치나 기기 변경 시 필요할 수 있으니 따로 보관해주세요.

설정 메뉴 하단의 라이선스 정보에서 상태, 라이선스키 일부, 인증일을 확인할 수 있습니다.

3. 홈 화면

홈 화면에서는 고시원 전체 상태를 한눈에 확인할 수 있습니다.

  • 전체 객실 수, 입주중 객실, 공실, 예약중
  • 월세연체, 수리중, 오늘 입주, 오늘 퇴실
  • 계약종료 임박

각 항목을 확인하면서 현재 고시원의 운영 상태를 빠르게 파악할 수 있습니다.

4. 객실 관리

객실 메뉴에서는 각 호실의 입주자 정보를 관리합니다.

  • 호실, 이름, 연락처
  • 입주일, 퇴실일
  • 월세, 보증금, 메모

입주자가 들어오면 해당 호실을 선택해 정보를 입력합니다. 입주자 정보가 변경되면 같은 화면에서 수정할 수 있습니다.

5. 월세 관리

월세 메뉴에서는 월별 납부 상태를 확인합니다.

  • 납부 예정 금액
  • 납부 여부와 미납 여부
  • 납부일과 월별 상태

납부가 완료된 입주자는 납부 처리해두면 미납 관리가 쉬워집니다.

6. 알림 설정

고시ONE 알림은 월세 납부 예정자를 미리 확인하기 위한 기능입니다.

예를 들어 납부일이 6월 14일이고 알림 시간이 오전 9시라면, 6월 13일 오전 9시에 알림이 옵니다.

알림 내용 예시: 202호 김철수님 월세 380,000원 납부 예정일이 내일입니다.

납부 예정자가 여러 명이면 사람별로 각각 알림이 올 수 있습니다. 알림을 누르면 고시ONE 앱이 열립니다.

7. 입퇴실 관리

입퇴실 메뉴에서는 입주 예정, 퇴실 예정, 공실, 퇴실 기록을 확인할 수 있습니다.

  1. 퇴실 예정 입주자를 확인합니다.
  2. 퇴실 처리 버튼을 누릅니다.
  3. 퇴실이 완료되면 해당 기록이 퇴실 기록에 남습니다.

퇴실 기록에는 호실, 이름, 연락처, 퇴실일, 입주일, 월세가 표시됩니다.

퇴실 기록은 자동 삭제되지 않습니다. 퇴실 후 1개월이 지난 기록에는 퇴실 후 1개월 경과 문구가 회색 작은 글씨로 표시됩니다.

필요 없는 퇴실 기록은 사용자가 직접 삭제 버튼으로 정리할 수 있습니다.

8. 민원/수리 관리

민원 또는 수리할 일이 있으면 민원/수리 메뉴에서 기록합니다.

  • 전등 교체, 도어락 문제, 수도 문제
  • 에어컨 점검, 벽지 보수

처리 전과 처리 완료 상태를 나누어 관리하면 놓치는 일이 줄어듭니다.

9. 비거주자/공용 관리

앱 설정에 따라 비거주자 또는 공용 요금 관리 기능을 사용할 수 있습니다.

공용 이용자, 별도 요금 납부자, 월세와 별도로 관리할 대상이 있을 때 이름, 연락처, 금액, 납부 상태를 입력해 관리합니다.

10. 백업

백업은 휴대폰 분실, 기기 변경, 앱 재설치에 대비하기 위해 꼭 필요합니다.

  • 데이터 내부 백업: 현재 휴대폰 안에 백업 저장
  • 데이터 외부 백업: 카카오톡, 파일 앱 등 외부 앱으로 백업 파일 공유

평소에는 내부 백업을 사용하고, 휴대폰을 바꾸기 전에는 반드시 외부 백업을 해두는 것을 추천합니다.

11. 복원

복원은 이전에 저장한 백업 파일을 다시 불러오는 기능입니다.

  • 내부 데이터 복원: 휴대폰 내부에 저장된 백업 복원
  • 외부 데이터 복원: 외부에서 받은 백업 파일을 선택해 복원
복원을 하면 현재 앱 안의 데이터가 백업 파일 내용으로 바뀔 수 있습니다. 복원 전에는 현재 데이터도 백업해두는 것이 좋습니다.

12. 설정

설정 메뉴에서는 앱 사용에 필요한 옵션을 관리합니다.

  • 알림 설정과 알림 시간
  • 백업/복원
  • 라이선스 정보
  • 앱 버전

설정 하단에는 앱 버전과 저작권 정보가 표시됩니다.

13. 자주 묻는 질문

알림이 오지 않아요.

휴대폰 설정에서 고시ONE 알림 권한이 켜져 있는지 확인해주세요. 또한 앱 설정에서 알림 기능과 알림 시간이 제대로 설정되어 있는지 확인해주세요.

앱을 삭제하면 데이터가 사라지나요?

앱 삭제 시 휴대폰 내부 데이터가 사라질 수 있습니다. 삭제 전에는 반드시 데이터 외부 백업을 해주세요.

휴대폰을 바꾸면 어떻게 하나요?

기존 휴대폰에서 데이터 외부 백업을 먼저 합니다. 새 휴대폰에 앱을 설치하고 라이선스 인증 후 외부 데이터 복원을 진행합니다.

퇴실 기록은 자동으로 없어지나요?

아니요. 퇴실 기록은 자동 삭제되지 않습니다. 필요 없는 기록은 직접 삭제 버튼으로 정리합니다.

라이선스키를 잃어버렸어요.

앱을 구매한 곳 또는 관리자에게 문의해주세요. 라이선스키는 재설치나 기기 변경 시 필요할 수 있으므로 꼭 보관하는 것이 좋습니다.

14. 사용 전 추천 순서

  1. 앱 설치
  2. 알림 권한 허용
  3. 라이선스 인증
  4. 설정에서 알림 시간 확인
  5. 객실 정보 입력
  6. 입주자 정보 입력
  7. 월세 납부일과 금액 확인
  8. 첫 백업 진행

처음에는 객실과 입주자 정보를 정확히 입력해두는 것이 가장 중요합니다. 기본 정보가 정확해야 월세, 알림, 입퇴실 관리가 편해집니다.
